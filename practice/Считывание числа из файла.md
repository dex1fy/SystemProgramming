## Программа для считывания числового значения из файла (с сохранением его в числовом формате). Исходный файл является бинарным (отдельно для Windows и Linux)

``` C
#define _CRT_SECURE_NO_WARNINGS
#include <stdio.h>
#define PATH "textBin.txt"

int main(void)
{
	FILE* myfile = fopen(PATH, "wb");
	double numberToWrite = 1555.656;
	fwrite(&numberToWrite, sizeof(double), 1, myfile);
	fclose(myfile);
	
	myfile = fopen(PATH, "rb");
	double numberToRead = 0;
	fread(&numberToRead, sizeof(double), 12, myfile);
	fclose(myfile);
	printf("%lf", numberToRead);
	return 0;
}
```
